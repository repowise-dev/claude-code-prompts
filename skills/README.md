# Skills Overview

This directory contains concise, task-focused skills for building and validating agent work.

## Included Skills

### `coding-agent-standards`
- File: `coding-agent-standards/SKILL.md`
- Focus: implementation discipline, low-risk edits, and clear delivery reporting.
- Use when: code changes need consistency, safety, and reviewable outcomes.

### `verification-agent`
- Files: `verification-agent/SKILL.md`, `verification-agent/strategies.md`
- Focus: risk-based verification workflow with explicit confidence reporting.
- Use when: validating code or prompt outputs before signoff.

### `prompt-architect`
- Files: `prompt-architect/SKILL.md`, `prompt-architect/reference.md`
- Focus: designing clear, testable prompts and improving weak prompts iteratively.
- Use when: authoring reusable prompt patterns or fixing prompt failures.

## Design Notes
- Each `SKILL.md` includes YAML frontmatter (`name`, `description`).
- Supporting docs are linked from each skill for progressive disclosure.
- Content is intentionally concise for fast loading and practical use.
